Instructions to run Problem 2a (Document Frequency):

Pre-requisites:
  - Hadoop must be installed and the `hadoop` command must be on PATH
    (or at /opt/homebrew/opt/hadoop/bin/hadoop).
  - The Wikipedia dataset archive must be available at:
      ~/Downloads/Assignment 2/Wikipedia-EN-20120601_ARTICLES.tar.gz
    (the script will auto-extract it into Data/ if needed).
  - Java 17 must be installed.

Step-1) Run Problem 2a (Document Frequency + Top-100 extraction):
        For the full dataset (10,000 articles):
        Run -> | bash scripts/run_problem2.sh full |

        For a quick test with 50 articles:
        Run -> | bash scripts/run_problem2.sh sample |

        This compiles the code, builds a fat JAR, and runs a two-job
        MapReduce pipeline:
          Job 1 — Computes Document Frequency (DF) for every stemmed term.
          Job 2 — Extracts the top 100 terms by DF using a min-heap.

        Output (full dataset):
          output/problem2a_df_full/part-r-00000     — Full DF listing (all terms)
          output/problem2a_top100_full/part-r-00000  — Top 100 terms by DF
          output/problem2a_df_full.tsv               — Same as above (convenience copy)
          output/problem2a_top100_full.tsv            — Same as above (convenience copy)

Instructions to run Problem 2b (TF-IDF Scorer):

Pre-requisites:
  - Hadoop must be running and accessible via the `hadoop` and `hdfs` commands.
  - The Wikipedia dataset must be uploaded to HDFS at:
      /user/$USER/input/wiki_full/Wikipedia-EN-20120601_ARTICLES
  - The stopwords.txt file must be uploaded to HDFS at:
      /user/$USER/stopwords.txt
  - Java 11 and Maven must be installed.

Step-1) Build the JAR file:
        Run -> | bash build.sh |
        This produces problem2.jar in the project root directory.

Step-2) Run Problem 2a — required input for 2b:
        Run -> | bash scripts/run_2a.sh |
        (see Problem 2a instructions above for details)

Step-3) Run Problem 2b (TF-IDF Scorer):
        Run -> | bash scripts/run_2b.sh |
        Output: output/tfidf_output.tsv
